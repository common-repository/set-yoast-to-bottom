Sets Yoast meta box priority as low, after custom fields.

== Description ==

This plugin sets the Yoast meta box priority as low, which places it after custom fields.

== Installation ==

1. Upload `set-yoast-to-bottom` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
